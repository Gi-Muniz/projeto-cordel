# Projeto Cordel

Projeto desenvolvido durante meus estudos iniciais de HTML e CSS, acompanhando uma aula do Curso em Vídeo, do professor Gustavo Guanabara.

A proposta foi praticar a criação de uma página web inspirada em literatura de cordel, trabalhando estrutura HTML, estilização com CSS, imagens de fundo e efeito visual com rolagem.

## Sobre o projeto

Este foi um dos meus primeiros contatos práticos com desenvolvimento front-end.

Durante o desenvolvimento, pratiquei a organização de uma página estática, aplicação de estilos, uso de fontes, imagens e efeitos visuais para criar uma experiência mais agradável ao usuário.

## Tecnologias utilizadas

* HTML5
* CSS3

## O que foi praticado

* Estruturação de página com HTML
* Estilização com CSS
* Uso de imagens de fundo
* Organização de conteúdo textual
* Aplicação de efeito parallax
* Primeiros passos em desenvolvimento web

## Status

Projeto finalizado como parte dos meus estudos iniciais em front-end.
